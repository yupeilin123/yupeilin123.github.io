如何在修改页面后，让页面自动刷新?

答案是使用 socket.io 。

socket.io 是一个 WebSocket 库，包括了客户端的 js 和服务器端的 nodejs ，它的目标是构建可以在不同浏览器和移动设备上使用的实时应用。它会自动根据浏览器从 WebSocket 、 AJAX 长轮询、 Iframe 流等等各种方式中选择最佳的方式来实现网络实时应用，非常方便和人性化，而且支持的浏览器最低达 IE5.5。

具体如何使用 socket.io 可以看官方文档。
